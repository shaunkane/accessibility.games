Plink → an audio game library

Plink is an open source library for creating audio games: action games, rhythm games, puzzle games, choose-your-own-adventure games, and whatever else you can come up with. Plink simplifies several difficult aspects about making audio games, such as finding sound effects, dealing with audio streaming, supporting diverse input methods. Plink games run in the browser and can be played on desktop or mobile.

Screen Racers → a game about screen readers

Screen Racers is a game that teaches players about how screen readers work. While screen readers are a commonly used assistive technology, it can be difficult for web developers to learn how to operate them. In Screen Racers, players navigate pages using a virtual screen reader and edit web pages to make them more screen reader accessible.
